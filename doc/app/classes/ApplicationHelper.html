date_select_tag(name, value = Date.today.to_s(:local), field_options = {}, img_options = {})

Requires the activecalendar plugin & javascript declarations. Creates a date selector for use inside a form_tag created form. You can change the defaults by passing standard html tag options to the field_options or img_options hashes, for example, you can change the clickable image to sdc.jpg like this: date_select("date", nil, img_options ={:src => ’/images/sdc.jpg’}
